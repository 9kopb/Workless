Elements

These are some of the built-in classes that you can use in your applications. Buttons, alerts, pagination, breadcrumbs, icons and more. Some items are automatic, such as syntax highlighting, just wrap your code in pre tags and it will be automatically formatted!

The best way to find out all of the available helpers is to take a look at the source and play with the framework, trying to list every possible option would be endless. You can mix and match items as well, such as adding an icon inside a button etc... This text also contains highlights and tooltips...give 'em a go!

Backgrounds

I've also decided to add a few repeatable background patterns in this version of Workless as I've been using them lots lately. so yeah. You can use them anywhere! I'm using them here is this demo for the body and footer.
