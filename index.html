What is Workless?

When designing/developing a new application we all have our own way of doing things. We all use certain frameworks, grid systems, plugins, code snippets etc...but what we do have in common is the fact that 99% of the time we encounter repetitive tasks that are the same on every project, enter Workless.

Workless aims to help you by offering a basic static HTML5 / CSS3 / Responsive base template filled with goodies to get you up and running on every project you do as quickly as possible.

At it's core Workless is simply a mashup of all the things I like about H5BP and Twitter Bootstrap with some additional extras which I tend to use on every project such as PrettyPhoto for lightboxes, Tipsy for tooltips, Prettify for syntax highlighting and various other bits and pieces.

I've removed a lot of stuff from both Bootstrap and H5BP to keep Workless lightweight and easy-to-use and also because it was stuff that I simply don't use/need.

Workless comes with responsive media queries setup for 'Mobile Portrait', 'Mobile Landscape', 'Tablet Portrait', 'Tablet Landscape/Small Browser' and 'Large Browser', you can demo the effect right now by simply scaling your browser window.

Remember, this isn't a 'Use this, its great!', it's simply a way I found to improve my workflow and cut down on repetitive tasks in every project. Feel free to use/abuse and hack into something that fits your workflow.

File Structure

Workless' file structure is very simple, checkout below:

workless
	index.html // this file
	assets/
		css/
			application.css // application specific styling
			plugins.css // styles for any plugins used
			workless.css // presets such as scaffolding and helpers
			responsive.css // responsive media queries
		img/ 
		js/
			application.js // application specific javascript
			jquery.js // jquery
			plugins.js // javascript for any plugins used

Usage

Using Workless is simple. You can either use the default index.html that ships with it as a template, or you can just include the CSS and JS in your own application:

CSS

<link href="assets/css/plugins.css" rel="stylesheet">
<link href="assets/css/workless.css" rel="stylesheet">
<link href="assets/css/application.css" rel="stylesheet">

Javascript

<!-- Grab Google CDN's jQuery, with a protocol relative URL; fall back to local if offline -->
<script src="//ajax.googleapis.com/ajax/libs/jquery/1.7.1/jquery.min.js"></script>
<script>window.jQuery || document.write('<script src="assets/js/jquery.js"><\/script>')</script>

<!-- scripts concatenated and minified via build script -->
<script src="assets/js/plugins.js"></script>
<script src="assets/js/application.js"></script>
<!-- end scripts -->
