What is Scaffolding?

Although I'm not a fan of grid systems in general I have included scaffolding to help you to build simple grid layouts of varying sizes.

The main reason I don't like grid systems is not the actual system, more the naming conventions used, .box1 .grid12 .row6 .span11..WTF!?!?

You'll notice the scaffolding included with Workless is much simpler with more semantic naming conventions, .one_full, .one_half, .one_third, .one_quarter. Of course this doesn't allow the flexibility of more elaborate grid systems, but I find it does what I need and at least I can remember the class names!
